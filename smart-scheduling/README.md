# 🚀 Smart Scheduling System

Sistema de agendamento moderno desenvolvido com Next.js, com foco em praticar desenvolvimento web e criar uma interface simples e funcional.

---

## 🛠️ Tecnologias utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS

---

## 💡 Funcionalidades

- 📅 Agendamento de horários
- 🧑 Inserção de nome do usuário
- ⏰ Seleção de data e horário
- ✅ Confirmação de agendamento

---

## ▶️ Como rodar o projeto

```bash
npm install
npm run dev
