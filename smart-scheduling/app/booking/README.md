# Smart Scheduling System

Sistema de agendamento moderno desenvolvido com Next.js.

## 🚀 Tecnologias
- Next.js
- React
- TypeScript
- Tailwind CSS

## 💡 Funcionalidades
- Agendamento de horários
- Interface simples e intuitiva
- Confirmação de agendamento

## ▶️ Como rodar o projeto

```bash
npm install
npm run dev
